# Truffle Hardhat

Hardhat integration to truffle project

## Dependencies

Install dependencies with `npm install` or `yarn`

## Compiling

```
npm run compile
yarn compile
```

## Testing

```
npm run test
yarn test
```

## Deploying

```
npm run deploy
yarn deploy
```
